6. 评论

   * url: '/article/write_comment'

   * method: post

   * params:

     * id //文章id
     * content: '' //评论内容
     * uid: ? //用户_id
     * type: 0 //0代表自己发布（即评论文章的），1代表评论别人的评论
     * toCommentId: 515 //如果是1的话，表明评论谁的

   * response: 

     ```json
     {
       ret: 0,
       message: '',
       data: 
     }
     ```
